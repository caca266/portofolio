Tentang Saya

Hai, saya Siti Anisa Saskia, seorang mahasiswi dari Universitas Nusa Putra, prodi Sistem Informasi.

Saya memiliki pengalaman dalam pengembangan web dan telah bekerja pada berbagai proyek. Saya sangat tertarik pada teknologi terbaru dan selalu berusaha untuk meningkatkan keterampilan saya.

Saya menguasai berbagai bahasa pemrograman dan teknologi seperti HTML, CSS, JavaScript.

Dalam analisis data, saya menggunakan alat seperti Python, R, dan SQL untuk membantu pengambilan keputusan yang lebih baik.

Sebagai manajer proyek, saya mampu merencanakan, mengorganisir, dan mengawasi proyek dari awal hingga selesai dengan efisien.

Meskipun masih mahasiswa, saya selalu berusaha menerapkan pengetahuan dan keterampilan dalam proyek nyata.
